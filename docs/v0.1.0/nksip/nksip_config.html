Description

NkSIP Config Server.

This module implements a simple ETS-based config store.

NkSIP has few global configuration options. Config values are set using standard nksip application's erlang environmet (file nksip.app). Their default values are defined in the following table. All of them can be changed at any time while running using put/2.

NameDefaultComments
timer_t1500Standar SIP T1 timer (msecs)
timer_t24000Standar SIP T2 timer (msecs)
timer_t45000Standar SIP T4 timer (msecs)
timer_c180Standar SIP C timer (secs)
ringing_timeout180 Time to allow for INVITE requests to answered (secs)
proxy_timeout600 Time a statefull proxy will wait for an answer (secs)
registrar_default_time3600(secs)
registrar_min_time60(secs)
registrar_max_time86400(secs)
max_connections1024 Maximum number of simultaneous TCP/TLS connections NkSIP will accept in each transport belonging to each SipApp.
tcp_timeout180 Time to disconnect TCP/SSL connection if no message has been received (secs)
dialog_timeout900 Time to destroy dialog if no message has been received (secs)
nonce_timeout30 Time a new nonce in an authenticate header will be usable (secs, only for ACK or requests coming from the same ip and port)

cseq/0

cseq() -> nksip:cseq()

Gets a new CSeq. After booting, CSeq's counter is set using nksip_lib:cseq/0. Then each call to this function increments the counter by one.
