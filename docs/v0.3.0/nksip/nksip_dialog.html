User Dialog Management Module.

Description

User Dialog Management Module. This module implements several utility functions related to dialogs.

id/1

id(Sipmsg::nksip:request() | nksip:response()) -> id()

Calculates a dialog's id from a nksip:request() or nksip:response(). Dialog ids are calculated as a hash over Call-ID, From tag and To Tag. If From tag and To tag are swapped the resulting id is the same.
